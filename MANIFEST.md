# Linux Assistant Manifest

Linux Assistant helps Linux users with daily workflows and administrative tasks in one consistent desktop app.

## Core mission

1. Provide practical, high-value Linux helper functionality without unnecessary complexity.
2. Support major Linux distributions and desktop environments with pragmatic compatibility.
3. Keep installation and operation simple, transparent, and maintainable.

## Product principles

1. **User-first utility**  
   Every workflow should solve a real user need (search, maintenance, diagnostics, updates, cleanup).
2. **Cross-distro pragmatism**  
   Distribution-specific behavior is handled where needed, while keeping shared logic centralized.
3. **Open ecosystem preference**  
   Free/open alternatives are preferred when they are mature and community-proven.
4. **Low operational friction**  
   Minimize required dependencies, reduce setup breakage, and provide clear fallbacks.
5. **Release reliability over churn**  
   Favor stable, testable releases and incremental improvements over risky rewrites.

## v0.7 direction

The v0.7 line focuses on hardening and maintainability:

1. Toolchain compatibility with current Flutter/Dart ecosystems.
2. Packaging and metadata consistency across build targets.
3. Better diagnostics and documentation for common environment issues.
4. Cleaner quality baseline for further feature delivery (v0.7.1 to v0.7.5).
